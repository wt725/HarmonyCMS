HarmonyCMS V1.0 内容管理系统
===============

> 运行环境要求PHP7.1+。

> HarmonyCMS 运行目录: public/

## 在线手册



## 安装

> 下载HarmonyCMS完整包解压到你本地（建议采用git方式拉取）

> 将你的站点绑定到public目录

> 将文件夹中HarmonyCMS.sql还原到你自己项目的数据库中（需自行创建数据库）

> 本机创建 .env 配置相关信息

> 访问后台并登录查看是否正常，后台目录为http://www.yourwebsite.com/admin

> 默认后台用户名：admin 密码:admin
